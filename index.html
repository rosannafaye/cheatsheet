HTML Cheatsheet

Table Tags
Attribute Name Description
<> Table The wrapper element for all HTML tables.
<thead> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows containing actual table data.
<tfoot> Table Foot The set of rows defining the footer in a table.
<tr> Table row The table row container.
<td> Table Data The table data container.
The Box Model
Attribute Name Description
<auto> Auto Margin The margin property will take the width of the element and will split the rest of the space equally between the left and right margins.
<margin-collapse> Margin Collapse Top and bottom margins of blocks are combined into a single margin equal to the largest individual block margin.
<overflow> Overflow Overflow can be set to hidden, or to scroll, which will make the overflowing content accessible via scroll bars within the original container.
<min-width> <min-height> Minimum Width/Minimum Height min-width and min-height properties can be used to set a minimum width and minimum height of an element’s box. CSS max-width and max-height properties can be used to set maximum widths and heights for element boxes.
<visibility> Visibility he CSS visibility property is used to render hidden objects invisible to the user, without removing them from the page.
<box-sizing> Box Model The property box-sizing controls which aspect of the box is determined by the height and width properties. The default value of this property is content-box, which renders the actual size of the element including the content box; but not the paddings and borders.
<border-box> Border Box The value border-box of the box-sizing property for an element corresponds directly to the element’s total rendered size, including padding and border with the height and width properties.
